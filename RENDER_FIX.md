# Исправление деплоя на Render

## Проблема
Ошибка: `bpm: команда не найдена` - в Render указана неправильная команда.

## Решение

### Вариант 1: Исправить в настройках Render (РЕКОМЕНДУЕТСЯ)

1. Зайдите на https://render.com
2. Откройте ваш сервис `lynchfm-radio`
3. Перейдите в **"Settings"**
4. Найдите **"Start Command"**
5. Измените на: `node server.js`
6. Нажмите **"Save Changes"**
7. Render автоматически перезапустит сервис

### Вариант 2: Использовать render.yaml

Файл `render.yaml` уже исправлен. Если используется автоматический деплой:
- Render автоматически подхватит изменения
- Или перезапустите деплой вручную

## Проверка

После исправления сервис должен запуститься и быть доступен по адресу:
`https://lynchfm-radio.onrender.com` (или ваш URL)

## Настройка переменных окружения

Убедитесь что в Render настроены:
- `NODE_ENV=production`
- `PORT=10000` (Render автоматически установит PORT, можно оставить пустым)
- `HOST=0.0.0.0`
- `ALLOWED_ORIGINS=https://your-netlify-site.netlify.app` (после деплоя на Netlify)
