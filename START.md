# Быстрый старт LynchFM

## Установка и запуск

1. **Установите Node.js** (если еще не установлен)
   - Скачайте с https://nodejs.org/

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Запустите сервер:**
   ```bash
   npm start
   ```

4. **Откройте браузер:**
   - Перейдите на http://localhost:3000

## Первое использование

### Для вещания:

1. Нажмите **"Подключиться к эфиру"**
2. Выберите источник звука:
   - Микрофон - для голоса
   - Системный звук - для музыки с компьютера
   - Музыкальный файл - для загрузки MP3/WAV файлов
3. Нажмите **"Начать трансляцию"**
4. Готово! Ваш звук транслируется

### Для прослушивания:

1. Откройте сайт в другом браузере/вкладке
2. Нажмите **"Слушать радио"**
3. Наслаждайтесь трансляцией!

## Советы

- Для захвата системного звука выберите вкладку с музыкой при запросе доступа
- Используйте Chrome или Edge для лучшей совместимости
- Звук транслируется только на радио, не воспроизводится локально при вещании
