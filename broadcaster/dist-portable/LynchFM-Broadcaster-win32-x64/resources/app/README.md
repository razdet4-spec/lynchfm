# LynchFM Broadcaster

Приложение для вещания на радиостанцию LynchFM.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

## Сборка для продакшена

### Windows
```bash
npm run build:win
```

### macOS
```bash
npm run build:mac
```

### Linux
```bash
npm run build:linux
```

Собранные приложения будут в папке `dist/`

## Использование

1. Запустите приложение
2. Введите URL сервера (например: https://yourdomain.com или http://localhost:3000)
3. Выберите источник звука:
   - **Системный звук** - захватывает звук с компьютера (выберите вкладку/окно с музыкой)
   - **Микрофон** - захватывает звук с микрофона
4. Нажмите "Подключиться"
5. Введите информацию о треке (опционально)
6. Начните вещание!

## Требования

- Node.js 18+
- Windows 10+, macOS 10.14+, или Linux

## Примечания

- Для захвата системного звука браузер попросит выбрать вкладку/окно
- Убедитесь что сервер LynchFM запущен перед подключением
- Рекомендуется использовать Chrome/Edge для лучшей совместимости
